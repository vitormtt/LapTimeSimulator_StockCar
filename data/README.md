# Pasta de Dados

Esta pasta contém dados de entrada e configurações para o simulador.

## Conteúdo

- Dados de pistas
- Configurações de veículos
- Parâmetros de simulação
- Dados de telemetria (se disponíveis)

## Formato de Dados

Os dados podem estar em diferentes formatos:
- CSV para dados tabulares
- JSON para configurações
- YAML para parâmetros
- Excel para planilhas complexas

## Exemplos

Futuramente, esta pasta conterá:
- `pistas/` - Dados de diferentes circuitos
- `veiculos/` - Especificações de veículos
- `config/` - Arquivos de configuração
